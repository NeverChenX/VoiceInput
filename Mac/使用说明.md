# VoiceInput 使用说明

## 📱 应用简介

VoiceInput 是一款 macOS 语音输入工具，按下快捷键即可录音，自动识别语音并粘贴文字到当前光标位置。

---

## 🚀 快速开始

### 1. 首次运行准备

在**系统设置 → 隐私与安全 → 辅助功能**中授权 VoiceInput，否则无法捕获键盘快捷键。

### 2. 启动应用

双击 `VoiceInput.app` 启动，状态悬浮窗将出现在屏幕右上角。

---

## ⌨️ 快捷键

| 操作 | 快捷键 | 说明 |
|------|--------|------|
| 开始录音 | `Option + Space` | 按住 Option，按空格开始录音 |
| 停止录音 | `Space` | 录音时按空格停止并识别 |
| 取消录音 | `ESC` | 录音时按 ESC 取消 |

---

## 🎨 界面说明

### 状态悬浮窗 (133×133 像素)

```
┌─────────────────┐
│    🔴 录音      │  状态指示区
│                 │
├─────────────────┤
───────────────   ← 分隔线
【自动发送】 [🟢] │  自动发送开关
───────────────   ← 分隔线
 Opt+Space    ⚙  │  快捷键提示 + 设置
└─────────────────┘
```

#### Zone A - 状态指示区（上方）
- **状态圆点**：显示当前状态颜色
  - 🔴 红色 = 录音中
  - 🔵 蓝色 = 识别中
  - ⚪ 灰色 = 待机
- **状态文字**：显示"录音"/"识别"/"待机"

#### Zone B - 自动发送开关（中间）
- **【自动发送】**：文字标签
- **开关**：点击切换开启/关闭
  - 🟢 绿色 = 开启，识别后自动按回车发送
  - ⚪ 灰色 = 关闭，仅粘贴不发送

#### Zone C - 快捷键 + 设置（底部）
- **快捷键提示**：显示当前开始录音的快捷键
- **⚙ 按钮**：点击打开设置对话框

### 全屏遮罩

录音或识别时，屏幕会显示半透明黑色遮罩：

| 状态 | 显示文字 |
|------|----------|
| 录音中 | "正在聆听" / "按 Space 停止" |
| 识别中 | "识别中" / "请稍候…" |
| 错误 | "错误" + 具体错误信息 |
| 已取消 | "已取消" |

## ⚙️ 设置

点击状态窗口右下角的 **⚙** 按钮打开设置对话框：

### 配置项

| 字段 | 说明 | 必填 |
|------|------|------|
| App ID | 火山引擎应用 ID | ✅ |
| Access Token | API 访问令牌 | ✅ |
| Secret Key | API 密钥 | ❌ |
| 开始录音快捷键 | 触发录音的组合键 | ❌ |
| 停止录音快捷键 | 停止录音的按键 | ❌ |
| 自动发送 | 识别后自动按回车 | ❌ |

### 快捷键捕获

1. 点击快捷键输入框
2. 按下想要的组合键（如 `Ctrl+Shift+A`）
3. 自动捕获并显示
4. 按 `ESC` 取消捕获

### 保存设置

- 点击**保存**按钮（或按回车）
- 配置立即生效，无需重启
- 设置保存到 `config.json` 文件

---

## 🔧 配置文件

配置文件 `config.json` 位于 `VoiceInput.app/Contents/MacOS/` 目录下：

```json
{
  "app_id": "你的应用ID",
  "access_token": "你的访问令牌",
  "secret_key": "你的密钥",
  "hotkey": "Option+Space",
  "hotkey_stop": "Space",
  "auto_enter": true
}
```

支持的快捷键格式：
- 修饰键：`Option`, `Command`, `Control`, `Shift`
- 功能键：`F1-F12`, `Space`, `Return`, `Tab`, `Esc`, `Delete`
- 字母键：`A-Z`
- 数字键：`0-9`

## 📝 使用方法

### 语音输入流程

1. **开始录音**：按 `Option + Space`
   - 屏幕显示"正在聆听"遮罩
   - 状态圆点变红色

2. **说话**：对着麦克风说话

3. **停止录音**：按 `Space`
   - 屏幕显示"识别中"遮罩
   - 状态圆点变蓝色

4. **自动粘贴**：识别完成后
   - 文字自动写入剪贴板
   - 模拟 `Cmd+V` 粘贴到光标位置
   - 如果开启"自动发送"，会再按回车

### 取消录音

录音时按 `ESC`，丢弃音频，不提交识别，显示"已取消"。

## 🖱️ 鼠标操作

- **拖拽窗口**：按住窗口任意位置拖动可移动
- **切换自动发送**：点击开关或中间区域
- **打开设置**：点击齿轮按钮

---

## ⚠️ 常见问题

### 无法启动或快捷键无效

1. 检查是否在**系统设置 → 隐私与安全 → 辅助功能**中授权了 VoiceInput
2. 重新启动应用

### 无法录音

1. 检查是否在**系统设置 → 隐私与安全 → 麦克风**中授权了 VoiceInput
2. 检查麦克风是否正常工作

### 识别失败

1. 检查 `config.json` 中的 `app_id` 和 `access_token` 是否正确
2. 检查网络连接
3. 查看日志文件 `voice_input.log`

---

## 📋 日志
